# 🎓 Rolê Universitário

Projeto acadêmico desenvolvido no curso de Análise e Desenvolvimento de Sistemas na FATEC, com o objetivo de propor uma solução digital para centralização de eventos universitários.

---

## 📌 Sobre o Projeto

O **Rolê Universitário** é uma proposta de aplicativo mobile que visa centralizar a divulgação de eventos acadêmicos e sociais, como palestras, festas, competições esportivas e atividades culturais.

Atualmente, essas informações encontram-se dispersas em diferentes canais (WhatsApp, Instagram, cartazes), dificultando o acesso dos estudantes. O projeto surge como uma solução para organizar e otimizar essa comunicação.

---

## 🎯 Objetivo

Desenvolver uma solução digital que:

- Centralize eventos universitários em um único ambiente  
- Facilite o acesso à informação  
- Aumente o engajamento dos estudantes  
- Melhore a comunicação dentro do campus  

---

## 👥 Público-Alvo

- 🎓 Estudantes universitários  
- 🏫 Atléticas e centros acadêmicos  
- 🏛️ Instituições de ensino  

---

## 🚀 Principais Funcionalidades

- 📅 Feed centralizado de eventos  
- 🔍 Filtros por tipo, curso e localização  
- 🔔 Notificações personalizadas  
- 🗺️ Mapa de calor de eventos  
- ❤️ Interação com eventos (curtir, comentar, confirmar presença)  
- 📤 Compartilhamento em redes sociais  

---

## 🧠 Diferenciais

- Centralização da informação  
- Experiência personalizada  
- Interface intuitiva  
- Engajamento social e acadêmico  
- Visualização em tempo real de movimentação no campus  

---

## 🏗️ Arquitetura Proposta

O sistema segue uma arquitetura em três camadas:

- **Front-end:** Aplicativo mobile e versão web  
- **Back-end:** API responsável pela lógica de negócio  
- **Banco de dados:** Armazenamento de usuários e eventos  

Comunicação via API REST com dados em JSON.

---

## ⚙️ Tecnologias Sugeridas

- **Mobile:** Flutter (Dart)  
- **Back-end:** Node.js (JavaScript/TypeScript)  
- **Banco de dados:** Firebase / PostgreSQL  
- **Prototipação:** Figma  
- **Versionamento:** Git + GitHub
